<h1 id="wallet-cli">Wallet CLI</h1>
<h2 id="usage">Usage</h2>
<p><code>./aeon-wallet-cli [options] [commands]</code></p>
<h2 id="description">Description</h2>
<p>This is the command line Aeon wallet. It is a tool to manage aeon
transfers and addresses. It needs to connect to an Aeon daemon to
work correctly.</p>
<h2 id="options">Options</h2>
<h3 id="-config"><code>--config</code></h3>
<p>Config file. Previously <code>--config-file</code>.</p>
<h3 id="-help"><code>--help</code></h3>
<p>Produce a help message with a list of available options.</p>
<h3 id="-log-file"><code>--log-file</code></h3>
<p>Specify log file.</p>
<h3 id="-log-file-size"><code>--log-file-size</code></h3>
<p>Specify maximum log file size [B]. Previously <code>--max-log-file-size</code>.</p>
<h3 id="-log-file-count"><code>--log-file-count</code></h3>
<p>Specify maximum number of rotated log files to be saved (no limit by setting to 0). Previously <code>--max-log-files</code>.</p>
<h3 id="-log-level"><code>--log-level</code></h3>
<p>0-4 or categories</p>
<h3 id="-node-address"><code>--node-address</code></h3>
<p>Use aeon daemon at ###host###:###port###. Previously <code>--daemon-address</code>.</p>
<pre><code>./aeon-wallet-cli --daemon-address=192.168.0.1:9149
</code></pre>
<h3 id="-node-ip"><code>--node-ip</code></h3>
<p>Use daemon instance at host arg instead of localhost. Previously <code>--node-ip</code>.</p>
<h3 id="-node-login"><code>--node-login</code></h3>
<p>Specify <code>username[:password]</code> for daemon RPC client. Previously <code>--daemon-login</code>.</p>
<h3 id="-node-no-relay"><code>--node-no-relay</code></h3>
<p>The newly created transaction will not be relayed to the Aeon network. Previously <code>--do-not-relay</code>.</p>
<h3 id="-node-port"><code>--node-port</code></h3>
<p>Use daemon instance at port arg instead of 11181. Previously <code>--daemon-port</code>.</p>
<h3 id="-node-trusted"><code>--node-trusted</code></h3>
<p>Enable commands which rely on a trusted daemon. Previously <code>--trusted-daemon</code>.</p>
<h3 id="-node-untrusted"><code>--node-untrusted</code></h3>
<p>Disable commands which rely on a trusted daemon. Previously <code>--untrusted-daemon.</code>.</p>
<h3 id="-node-version-conflict"><code>--node-version-conflict</code></h3>
<p>Allow communicating with a daemon that uses a different RPC version. Previously <code>--allow-mismatched-daemon-version</code>.</p>
<h3 id="-restore-from-seed"><code>--restore-from-seed</code></h3>
<p>Recover wallet using Electrum-style mnemonic seed. Previously <code>--restore-deterministic-wallet</code>.</p>
<h3 id="-restore-from-seed-multisig"><code>--restore-from-seed-multisig</code></h3>
<p>Recover multisig wallet using Electrum-style mnemonic seed.Previously <code>--restore-multisig-wallet</code>.</p>
<h3 id="-wallet"><code>--wallet</code></h3>
<p>Generate new wallet and save it to arg. Previously <code>--generate-new-wallet</code>.</p>
<h3 id="-wallet-address-file"><code>--wallet-address-file</code></h3>
<p>Create an address file for new wallets. Previously <code>--create-address-file</code>.</p>
<h3 id="-wallet-from-file"><code>--wallet-from-file</code></h3>
<p>Use wallet Previously <code>--wallet-file</code>.</p>
<h3 id="-wallet-from-json"><code>--wallet-from-json</code></h3>
<p>Generate wallet from JSON format file. Previously <code>--generate-from-json</code>.</p>
<h3 id="-wallet-from-keys"><code>--wallet-from-keys</code></h3>
<p>Generate wallet from private keys Previously <code>--generate-from-keys</code>.</p>
<h3 id="-wallet-from-seed"><code>--wallet-from-seed</code></h3>
<p>Specify Electrum seed for wallet recovery/creation. Previously <code>--electrum-seed</code>.</p>
<h3 id="-wallet-from-spend-key"><code>--wallet-from-spend-key</code></h3>
<p>Generate deterministic wallet from spend key. Previously <code>--generate-from-spend-key</code>.</p>
<h3 id="-wallet-from-view-key"><code>--wallet-from-view-key</code></h3>
<p>Generate incoming-only wallet from view key. Previously <code>--generate-from-view-key</code>.</p>
<h3 id="-wallet-height"><code>--wallet-height</code></h3>
<p>Restore from specific blockchain height. Previously <code>--restore-height</code>.</p>
<h3 id="-wallet-language"><code>--wallet-language</code></h3>
<p>Language for mnemonic. Previously <code>--mnemonic-language</code>.</p>
<h3 id="-wallet-non-deterministic"><code>--wallet-non-deterministic</code></h3>
<p>Generate non-deterministic view and spend keys</p>
<h3 id="-wallet-multisig"><code>--wallet-multisig</code></h3>
<p>Generate a master wallet from multisig wallet keys Previously <code>--generate-from-multisig-keys</code>.</p>
<h3 id="-wallet-password"><code>--wallet-password</code></h3>
<p>Wallet password (escape/quote as needed). Previously <code>--password</code>.</p>
<h3 id="-wallet-password-file"><code>--wallet-password-file</code></h3>
<p>Wallet password file. Previously <code>--password-file</code>.</p>
<h3 id="-wallet-ringdb-dir"><code>--wallet-ringdb-dir</code></h3>
<p>Set shared ring database path. Previously <code>--shared-ringdb-dir</code>.</p>
<h3 id="other">Other</h3>
<h4 id="-english"><code>--english</code></h4>
<p>Display English language names. Previously <code>--use-english-language-names</code>.</p>
<h4 id="-key-rounds"><code>--key-rounds</code></h4>
<p>Number of rounds for the key derivation function. Previously <code>--kdf-rounds</code>.</p>
<h4 id="-limit-threads"><code>--limit-threads</code></h4>
<p>Max number of threads to use for a parallel job. Previously <code>--max-concurrency</code>.</p>
<h4 id="-notify-tx"><code>--notify-tx</code></h4>
<p>Run a program for each new incoming transaction, '%s' will be replaced by the transaction hash. Previously <code>--tx-notify</code>.</p>
<h4 id="-stagenet"><code>--stagenet</code></h4>
<p>For stagenet. Daemon must also be launched with --stagenet flag</p>
<h4 id="-subaddress-lookahead"><code>--subaddress-lookahead</code></h4>
<p>Set subaddress lookahead sizes to major:minor</p>
<h4 id="-testnet"><code>--testnet</code></h4>
<p>For testnet. Daemon must also be launched with --testnet flag</p>
<h4 id="-version"><code>--version</code></h4>
<p>Output version information ###Aeon 'Chronos' (v0.14.0.0-release)###</p>
